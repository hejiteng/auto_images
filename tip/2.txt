> 2. **全局替换 Fabric API 为 Konva API**：
>
>    | Fabric.js | Konva |
>    |-----------|-------|
>    | `new fabric.Canvas(el)` | `new Konva.Stage({ container: el, width, height })` + `new Konva.Layer()` |
>    | `fabric.Image.fromURL(url, cb)` | `Konva.Image` + `new Image()` 加载 |
>    | `new fabric.Circle({...})` | `new Konva.Circle({...})` |
>    | `new fabric.Rect({...})` | `new Konva.Rect({...})` |
>    | `new fabric.Text({...})` | `new Konva.Text({...})` |
>    | `canvas.add(obj)` | `layer.add(obj)` |
>    | `canvas.remove(obj)` | `obj.destroy()` 或 `layer.remove(obj)` |
>    | `canvas.renderAll()` | `layer.batchDraw()` 或 `layer.draw()` |
>    | `canvas.setDimensions({w,h})` | `stage.size({ width, height })` |
>    | `canvas.setBackgroundImage(img, cb)` | `layer.add(new Konva.Image({ image: img, ... }))` |
>    | `canvas.getPointer(e)` | `stage.getPointerPosition()` |
>    | `canvas.getWidth()` / `getHeight()` | `stage.width()` / `stage.height()` |
>    | `canvas.clear()` | `layer.destroyChildren()` |
>    | `canvas.dispose()` | `stage.destroy()` |
>    | `obj.set({ key: val })` | `obj.setAttr('key', val)` 或 `obj.key(val)` |
>    | `obj.get('key')` | `obj.getAttr('key')` 或 `obj.key()` |
>

## 关键差异提醒

**模板变化**：
```vue
<!-- 之前 -->
<canvas ref="canvasEl"></canvas>

<!-- 之后 -->
<div ref="containerEl" class="canvas-container"></div>
```

**初始化变化**：
```javascript
// 之前
canvas = new fabric.Canvas(canvasEl.value)

// 之后
const stage = new Konva.Stage({
  container: containerEl.value,
  width: 800,
  height: 400,
})
const layer = new Konva.Layer()
stage.add(layer)
```

**背景图变化**：
```javascript
// 之前
fabric.Image.fromURL(url, (img) => {
  canvas.setBackgroundImage(img, ...)
})

// 之后
const imageObj = new Image()
imageObj.src = url
imageObj.onload = () => {
  const konvaImg = new Konva.Image({
    image: imageObj,
    width: w,
    height: h,
  })
  layer.add(konvaImg)
  layer.draw()
}
```
